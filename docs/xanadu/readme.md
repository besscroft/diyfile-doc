# 项目介绍

一款好看的在线网盘程序，后端基于 Spring Boot 3 构建，前端基于 Vitesse 构建。


> 本项目在不断更新维护，更多组件也会在后续的版本中开发迭代。虽然开发时间可能会很长，但是我们会尽可能地提供更多的功能、更完善的架构、更好的解决方案，以便您能够更好的使用。
